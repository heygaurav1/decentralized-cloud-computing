# Decentralized-Cloud-Computing

## Project Description  
Decentralized Cloud Computing is a Soroban smart contract built on the Stellar network that enables a trustless marketplace for distributed compute resources.  

It allows independent providers to offer computing power while clients submit jobs that are coordinated on-chain and executed off-chain. The system ensures transparency, traceability, and secure coordination without relying on centralized cloud providers.

---

## ⚙️ What it Does  
- Registers compute providers with pricing  
- Allows clients to submit compute jobs  
- Assigns jobs to providers  
- Tracks job lifecycle on-chain  
- Maintains transparent and verifiable records  

---

## ✨ Features  
- 🖥 **Provider Registration** — Register with price per compute unit  
- 📦 **Job Submission** — Clients submit compute workloads  
- 🔐 **Secure Authorization** — Wallet-based authentication  
- 📊 **On-chain Tracking** — Transparent job lifecycle  
- ✅ **Completion Verification** — Providers finalize jobs  
- 🔍 **Queryable State** — Fetch jobs and provider details  
- ⚡ **Lightweight & Efficient** — Built using Soroban  

---

## 🏗 Architecture Overview  
1. Providers register and define pricing  
2. Clients submit compute jobs  
3. Jobs are assigned to providers  
4. Execution happens off-chain  
5. Providers mark jobs completed on-chain
